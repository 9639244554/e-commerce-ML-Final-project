# Customer Satisfaction Prediction using Statistics and Machine Learning

## 📌 Project Overview

This project analyzes customer satisfaction using both statistical hypothesis testing and machine learning techniques. Statistical tests were used to identify significant relationships between customer attributes and satisfaction, followed by predictive modeling using multiple classification algorithms.

---

## 🎯 Objectives

- Analyze customer behavior using statistical methods.
- Test hypotheses about customer satisfaction.
- Build machine learning models to predict customer satisfaction.
- Compare model performance and select the best model.

---

## 📂 Dataset Features

- Age
- Gender
- City
- Membership Type
- Total Spend
- Items Purchased
- Average Rating
- Discount Applied
- Days Since Last Purchase
- Satisfaction Level (Target)

---

## 📊 Statistical Analysis

The following statistical techniques were applied:

- Descriptive Statistics
- Data Distribution Analysis
- Normality Testing
- Hypothesis Testing
- P-value Interpretation
- Chi-Square Test
- Independent T-Test
- One-Way ANOVA
- Mann–Whitney U Test

These analyses helped determine which customer attributes have statistically significant relationships with customer satisfaction.

---

## 🤖 Machine Learning Models

- Random Forest
- Logistic Regression
- Support Vector Machine (SVM)
- Decision Tree

---

## 📈 Model Performance

| Model | Accuracy |
|--------|----------|
| Random Forest | **98.57%** |
| Logistic Regression | **98.57%** |
| Support Vector Machine | **98.57%** |
| Decision Tree | **97.14%** |
